## Endpoint Quick Access

### All 13 Endpoints at a Glance

**Posts (4)**
| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/create_post` | POST | Create post |
| `/get-all-post` | POST | Get all posts |
| `/get-posts` | POST | Get posts (paginated) |
| `/delete_post/{postid}` | POST | Delete post |

**Comments (3)**
| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/Comment` | POST | Create comment |
| `/comment_bypost` | GET | Get comments for post |
| `/delete_comment` | DELETE | Delete comment |

**Likes (2)**
| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/Like` | POST | Like post/comment |
| `/like_bypost` | GET | Get likes for post |

**Friends/Users (3)**
| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/add_friend` | POST | Add friend |
| `/allusers` | GET | List all users |
| `/action_by_user_id` | GET | Get user actions |

**Blocking (1)**
| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/block_user` | POST | Block user |
